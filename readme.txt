== Description ==

This plugin uses shortcode `[blog_activity]` to display a forum like table of your blog activity into your page or post.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `BlogActivityShortcode.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[blog_activity]` in a page or post.
